# 📱 AOSP ROMs -- Fastboot Installer

Custom **AOSP-based ROMs** installable via **fastboot**

------------------------------------------------------------------------

## ✨ Features

-   🔄 **Boot Selector**

    -   Choose between **Root** or **No Root**
    -   Apatch password:
        -   `aosp1234`
        -   `redminote12s`

-   📂 **EXT4 File System**

    -   All partitions (including `super`) use **EXT4**

-   🔓 **Full Read/Write (R/W) Access**

-   ⚡ **Fastboot Installer**

    -   Simple and automatic flashing process

-   🛡️ **VBMeta Disabled by Default**

-   🔓 **Bootloader Compatibility**

    -   Works with:
        -   Official unlock via Mi Unlock Tool
        -   Unofficial unlock via MTK Client

------------------------------------------------------------------------

## ⚠️ IMPORTANT WARNING

🚨 **THIS INSTALLATION WILL ERASE ALL YOUR DATA.**\
All apps, photos, files, and internal storage will be permanently
deleted.

Make sure to create a full backup before proceeding.

------------------------------------------------------------------------

## 📦 Installation

1.  Open the provided `.bat` file.
2.  Select the desired boot option (Root or No Root).
3.  The installation will start automatically.

Do not disconnect the device during the process.
